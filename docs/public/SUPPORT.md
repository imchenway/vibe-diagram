# Vibe Diagram 支持

## 支持入口

请通过 GitHub Issues 报告缺陷、安装问题、安全边界问题或文档缺口：`https://github.com/imchenway/vibe-diagram/issues`。

提交问题时请提供：

- Codex App 或 CLI 的版本和操作系统；
- 安装方式与固定版本标签；
- 可复现的提示词或最小输入；
- 实际结果、预期结果和错误输出；
- 已移除密钥、个人信息和私有路径的相关日志。

## 支持范围

公开证据只覆盖兼容性账本中明确标记为通过的范围。未验证客户端、操作系统、升级路径或 UI 行为不会被描述为已支持。

安全问题如果不适合公开披露，请先创建不包含利用细节和秘密信息的最小 GitHub Issue，请求私下沟通方式。
