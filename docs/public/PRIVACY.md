# Vibe Diagram 隐私政策

生效日期：2026-07-17。

## 适用范围

本政策适用于开源项目 `Vibe Diagram` 提供的 agent skill 和 skills-only plugin。该项目帮助用户依据提示词、文件和仓库证据生成自包含 HTML 图。

## 数据处理

发布者不会运营后端服务，不会通过本项目收集、传输、出售或持久化用户的提示词、文件内容、生成结果或个人信息。本项目不包含账号系统、分析代码、广告、Cookie 或远程遥测。

安装并调用 skill 后，宿主应用可能根据用户授权读取当前工作区文件，并在用户环境中生成 HTML 文件。具体数据如何由宿主应用处理，适用该宿主应用和 OpenAI 的相应政策；发布者无法访问宿主应用内部的数据。

## 本地脚本与输出

本项目包含仅使用 Python 标准库的本地 HTML 检查脚本。该脚本用于验证自包含资源、模板身份和结构约束，不执行网络请求。生成文件由用户和宿主环境自行保存、共享或删除。

## 第三方服务

项目文档可能链接到 GitHub、OpenAI 或其他宿主产品。访问这些服务时，适用对应服务的隐私政策。

## 联系方式

隐私问题可通过 GitHub Issues 提交：`https://github.com/imchenway/vibe-diagram/issues`。

## 变更

如果项目的数据处理方式发生变化，发布者将在公开仓库中更新本政策及其生效日期。
